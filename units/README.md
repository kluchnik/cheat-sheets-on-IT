# Единицы измерения

ГОСТ 8.417-2002 (Единицы величин)  
| Наименование | Обозначение (рус) | Обозначение (eng) |
| - | - | - |
| бит | бит | bit |
| байт | Б (байт) | B (byte) |

1 Б = 8 бит  

Также может встречатся:
| Наименование | Обозначение (рус) | Обозначение (eng) |
| - | - | - |
| бит | б (бит) | b (bit) |
| байт | Б (байт) | B (byte) |
